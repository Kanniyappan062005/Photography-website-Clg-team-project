# 📸 Photography Website

A visually stunning and responsive photography website designed to showcase beautiful images, portfolios, and creative work. Built with modern web technologies, this website offers an elegant and user-friendly experience.

## 🌟 Features

- 📷 **Gallery Display** – Showcase high-quality images in a grid or slider format.  
- 🎨 **Clean & Minimal UI** – Aesthetic design with a focus on visuals.  
- 📱 **Fully Responsive** – Optimized for desktops, tablets, and mobile devices.  
- 🌙 **Dark & Light Mode** – Toggle between themes for a better user experience.  
- 🔍 **SEO Optimized** – Well-structured for search engines.  
- ⚡ **Fast Loading** – Optimized images and performance enhancements.  
- 📨 **Contact Form** – Visitors can easily reach out.  
- 🔗 **Social Media Links** – Connect with your audience via Instagram, Facebook, etc.

## 🚀 Technologies Used

- **HTML5, CSS3, JavaScript** – Core front-end technologies.  
- **Bootstrap / Tailwind CSS** – For responsive and modern styling.  
- **GSAP / AOS.js** – Smooth animations and scrolling effects.  
- **Email.js / Formspree** – Contact form integration.  
- **Git & GitHub** – Version control and hosting.
